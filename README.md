# 개발 목표

1.	날씨에 맞는 스타일 추천으로 사용자의 관심을 유도한다. 
2.	스타일 속 아이템의 정보를 제공하여 쇼핑으로의 접근성을 확대한다.
3.	커뮤니티를 통한 사용자간 정보 공유로 플랫폼을 활성화한다. 
4.	태그를 이용하여 손쉬운 스타일의 정보를 검색 가능하게 한다.
